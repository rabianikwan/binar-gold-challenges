# How To Run
Install all dependency by:
```
pnpm install
```
copy or create .env file
```
cp env.example .env
```
config .env file(example) :

```env
PORT=5000
NODE_ENV=development
DB_USER=root
DB_PASS=root
DB_NAME=binar
```
run project by : 
```
pnpm dev or pnpm start
```
